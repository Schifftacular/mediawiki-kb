# Summary Page Template

## Page: Projects/{Name}/Summary

### Prerequisites
None — this is typically the first page to fill out.

### What To Generate
- One-paragraph executive summary of the venture
- Mission statement (why this exists)
- Vision statement (where this is going)
- Core value proposition (what makes this different)
- Target market overview (who this serves)
- Current stage and key milestones
- Founding team overview

### Prompting Framework
Start with the simplest question: "Explain {name} to someone who's never heard of it, in two sentences." Build outward from there. Use the Elevator Pitch Framework: Problem → Solution → Why Us → Why Now. After the pitch is clear, expand into mission/vision using the Golden Circle: Why → How → What.

### Session Type
Claude Desktop or Cowork — this is a foundational conversational task. The user needs to articulate their vision clearly. Code is not involved.

### Downstream Effects
After creating or editing this page, evaluate ALL other pages in the project — Summary is the foundational page that everything else builds on. Pay special attention to:
- Brand Vision (must align with mission/vision)
- Product Vision (must align with value proposition)
- Target Audience (must align with market overview)
- Positioning (must align with differentiation)
